Bootstrap Exercise

Read me:

As an assignment we had to get familiar with bootstrap, an opensource collection of resources to help you build a website with pre-made css-styling, java-script, everything prepared for you! It makes your website kind of look generic because there are only a few styles to chose from but I kind of liked it, it made my website responsive in no time andfound it easy to implement some stuff that would take me a long time to build from scratch. It has its limitations but for project one-pagers or promotional use I can totally see the benefit of building a web page in Bootstrap!
