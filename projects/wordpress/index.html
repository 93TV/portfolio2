WordPress

As a student web developer we had to make a WordPress site. This was one of the assignments that I liked less than the others. WordPress is a very robust and slow tool to work with, a lot of times you're losing time just because you're searching for a way to edit a small part of the website... Although this struggle I totally understand the reason of this assignment, a majority of websites are build with WordPress and the chance that we'll one day need to edit or maintain a WordPress website in our career is plausible.
